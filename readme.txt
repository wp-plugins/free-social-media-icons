Adds Email, RSS, Facebook, Twitter and other social follow icons to your blog.

== Description ==

This plugin helps you to connect with your readers on the most popular social media platforms and via Email, the most effective way to retain readers.

The plugin is very simple as compared to other social media plugins. It includes Email, RSS, Facebook, Twitter, Flickr, Linkedin, Youtube and other popular social media follow icons. The plugin comes with 40+ default icon designs, but you can also upload other icon designs. 

The icons are displayed in a widget and you can place this widget on any page of your blog. You can select the icon size, order and type of display (horizontally, vertically) and other display options.

== Installation ==

1. Upload `free-social-media-icons.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Define your icon settings on the left main menu bar.
4. Go to the “Appearance” section on the left main menu bar and drag the widget to your preferred destination.
